                      ## 🧬 What Is This?

                      A collection of **72 autonomous Claude AI skills** for communication students — covering six departments across the entire Faculty of Communication curriculum.

                      Each skill transforms Claude into an **agentic domain expert** that researches the web, reads your project files, creates production-ready deliverables, self-reviews its own output, and iterates — not just advises.

                      > [!TIP]
                      > > **How it works:** Each skill is a structured `.md` prompt file. Drop it into a Claude Project or Claude Code, and Claude instantly becomes your creative production expert — with real industry frameworks, professional output templates, and deep domain knowledge.
                      > >
                      > > <br>

                      ## ⚡ What Makes These Skills Different?

                      These aren't chatbot personas. They're **autonomous agents** that execute a full research → create → review cycle:

                      | | Capability | Description |
                      |:---:|:---|:---|
                      | 🔍 | **Research First** | Search the web for current data, trends, and benchmarks before creating anything |
                      | 📂 | **Read Project Context** | Analyze existing files, briefs, and guidelines in your project |
                      | 📝 | **Create Real Deliverables** | Write production-ready files (scripts, briefs, reports, plans) directly to your project |
                      | 🔁 | **Self-Review** | Re-read their own output and assess quality before presenting |
                      | 🎯 | **Iterate** | Offer specific refinement directions after every deliverable |

                      ## 🚀 Quick Start

                      ### Option 1 — Claude Projects *(Recommended)*

                      ```
                      1. Go to claude.ai → Projects → Create Project
                      2. Upload SKILL.md files from your department folder into the Knowledge section
                      3. Start chatting — Claude now has your skills loaded
                      ```

                      ### Option 2 — Claude Code CLI

                      ```bash
                      git clone https://github.com/AlterLab-IEU/AlterLab-FC-Skills.git
                      cd AlterLab-FC-Skills
                      claude "act as my screenwriter assistant and help me develop a short film concept"
                      ```

                      ### Option 3 — Pick Individual Skills

                      > Browse the [`skills/`](skills/) folder and download only the ones you need. Every skill is a standalone `.md` file.
                      >

                      ## 🔬 How Skills Work
                   
                      Each skill is a structured prompt in Markdown format. When loaded into a Claude Project or used with Claude Code CLI, it transforms Claude into an **autonomous agent** with:
                   
                      | | Feature | Details |
                      |:---:|:---|:---|
                      | 🤖 | **Agentic Execution** | Skills research, create files, and iterate autonomously |
                      | 🏗️ | **Industry Frameworks** | Real methodologies — ROSTIR, PESO, RACE, AIDA, Barcelona Principles |
                      | 📄 | **File-Based Deliverables** | Production-ready outputs written directly to your project |
                      | 🔄 | **Self-Review Loops** | Agents assess their own work before presenting it |
                      | 🌐 | **Research-Driven Workflow** | Web search for current data before every deliverable |
                      | 🎨 | **Craft Principles** | Opinionated expert knowledge, not generic advice |
                   
                      > [!NOTE]
                      > > **Agentic Workflow:** Every skill follows this autonomous cycle — **Research** (search web for current data) → **Context** (read existing project files) → **Create** (write structured deliverable) → **Review** (assess quality & completeness) → **Present** (summarize + offer 3 next paths).
                      > >
                      > > <br>
