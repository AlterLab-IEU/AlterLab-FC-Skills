# 🏗️ AlterLab Foundation Codebase

**Shared utilities, configs & boilerplate for AlterLab projects**

> The foundation codebase that powers AlterLab Creative Technologies Laboratory projects at İzmir University of Economics — providing shared configurations, utilities, and boilerplate code across all lab repositories.
>
> ---
>
> ## ✨ Overview
>
> AlterLab FC serves as the foundational layer for all AlterLab projects, ensuring consistency and code reuse across the laboratory's diverse portfolio of AI-powered applications, interactive experiences, and educational tools.
>
> ## 🧩 What's Included
>
> - **Shared Configurations** — Common project settings and environment templates
> - - **Utility Functions** — Reusable helper code across AlterLab projects
>   - - **Boilerplate Templates** — Starter code for new lab projects
>     - - **Development Standards** — Coding conventions and project structure guidelines
