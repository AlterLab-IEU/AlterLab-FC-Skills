# AlterLab FC Skills

> **48 Agentic Claude AI skills for Faculty of Communication students**
> Built by [AlterLab](https://github.com/AlterLab-IEU) — Creative Technologies Laboratory

## What Is This?

A collection of **48 autonomous Claude AI skills** for communication students — covering Public Relations & Advertising, Cinema & Digital Media, New Media & Communication, and Generative AI Production. Each skill transforms Claude into an **agentic domain expert** that researches, creates file-based deliverables, self-reviews, and iterates — not just advises.

**Not tied to any specific university.** These skills work for any communication student, anywhere.

---

## What Makes These Skills Different?

These aren't chatbot personas. They're **autonomous agents** that:

- **Research first** — Search the web for current data, trends, and benchmarks before creating anything
- **Read project context** — Analyze existing files, briefs, and guidelines in your project
- **Create real deliverables** — Write production-ready files (scripts, briefs, reports, plans) directly to your project
- **Self-review** — Re-read their own output and assess quality before presenting
- **Iterate** — Offer specific refinement directions after every deliverable

## Quick Start

### Option 1 — Claude Projects (Recommended for students)

1. Go to [claude.ai](https://claude.ai) → **Projects** → **Create Project**
2. Upload the `SKILL.md` files from your department folder into the project's **Knowledge** section
3. Start chatting — Claude now has your skills loaded

### Option 2 — Claude Code CLI

```bash
git clone https://github.com/AlterLab-IEU/AlterLab-FC-Skills.git
cd AlterLab-FC-Skills
claude "act as my screenwriter assistant and help me develop a short film concept"
```

### Option 3 — Pick Individual Skills

Browse the `skills/` folder and download only the ones you need. Every skill is a standalone `.md` file.

## How Skills Work

Each skill is a structured prompt in Markdown format. When loaded into a Claude Project or used with Claude Code CLI, it transforms Claude into an **autonomous agent** with:

- **Agentic execution** — Skills research, create files, and iterate autonomously
- **Industry frameworks** — Real methodologies (ROSTIR, PESO, RACE, AIDA, Barcelona Principles)
- **File-based deliverables** — Production-ready outputs written directly to your project
- **Self-review loops** — Agents assess their own work before presenting it
- **Research-driven workflow** — Web search for current data before every deliverable
- **Craft principles** — Opinionated expert knowledge, not generic advice

### Agentic Workflow

Every skill follows this autonomous cycle:

```
1. Research  → Search web for current data, trends, benchmarks
2. Context   → Read existing project files, briefs, guidelines
3. Create    → Write deliverable as a structured markdown file
4. Review    → Re-read output, assess quality and completeness
5. Present   → Summarize key decisions, offer 3 refinement paths
```
